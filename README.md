# 📚 Tecnico-em-Desenvolvimento-de-Sistemas

Repositório destinado ao armazenamento de atividades, projetos e estudos desenvolvidos ao longo do curso **Técnico em Desenvolvimento de Sistemas** pelo SENAI.

---

## 📌 Sobre o repositório

Este repositório tem como objetivo organizar e documentar conteúdos aprendidos durante a formação técnica, reunindo exercícios práticos, projetos e materiais de apoio que contribuem para o desenvolvimento de habilidades na área de tecnologia.

Aqui você encontrará desde fundamentos básicos até aplicações mais estruturadas, demonstrando a evolução do aprendizado ao longo do curso.

---

## 🧠 Atividades desenvolvidas

As atividades presentes neste repositório incluem:

- Exercícios de lógica de programação  
- Desenvolvimento de algoritmos  
- Projetos práticos individuais e em grupo  
- Simulações de situações reais do mercado de trabalho  

Cada atividade foi elaborada com foco no aprendizado progressivo e na aplicação prática dos conceitos estudados.

---

## 💡 Lógica de programação e algoritmos

Uma das bases fundamentais deste curso é a **lógica de programação**, onde são trabalhados conceitos como:

- Estruturas condicionais (if/else)  
- Estruturas de repetição (for, while)  
- Variáveis e tipos de dados  
- Vetores e matrizes  
- Funções e modularização  

Os algoritmos desenvolvidos visam resolver problemas de forma eficiente, organizada e lógica, preparando para linguagens de programação mais avançadas.

---

## 🌐 Arquitetura de redes e IoT

O repositório também contempla estudos relacionados à **arquitetura de redes** e à **Internet das Coisas (IoT)**, incluindo:

- Conceitos básicos de redes de computadores  
- Modelos de comunicação (cliente-servidor)  
- Protocolos de rede  
- Integração de dispositivos inteligentes  
- Aplicações de IoT no dia a dia  

Esses conteúdos são essenciais para entender como sistemas se comunicam e como dispositivos podem ser conectados de forma inteligente.

---

## 📋 Levantamento de requisitos

Outro ponto importante abordado é o **levantamento de requisitos**, etapa fundamental no desenvolvimento de sistemas. Aqui são trabalhados:

- Identificação das necessidades do cliente  
- Requisitos funcionais e não funcionais  
- Documentação de sistemas  
- Análise e validação de requisitos  

Essa prática garante que o sistema desenvolvido atenda corretamente às expectativas e objetivos propostos.

---

## 💻 Sistemas Operacionais

O estudo de **sistemas operacionais** também faz parte da formação, abordando:

- Conceitos básicos de funcionamento  
- Gerenciamento de processos  
- Sistemas de arquivos  
- Interface com o usuário  
- Uso prático de sistemas operacionais no desenvolvimento  

Esse conhecimento é essencial para compreender o ambiente onde os sistemas e aplicações são executados.

---

## 🚀 Objetivo

Este repositório tem como principal objetivo:

- Consolidar o aprendizado adquirido no curso  
- Servir como portfólio de projetos e atividades  
- Auxiliar na revisão de conteúdos  
- Demonstrar habilidades técnicas na área de TI
